# MEATZE Campus

MEATZE Campus is a Django-based educational platform designed for managing students, teachers and administrative workflows.

## Features
- Student, teacher and admin panels
- Course and schedule management
- Role-based authentication
- AI assistant integration
- WhatsApp Cloud API integration
- REST API (Django Rest Framework)

## Tech Stack
- Python 3
- Django
- PostgreSQL
- Django REST Framework
- Node.js (frontend assets)
